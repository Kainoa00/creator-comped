Design a marketing WEBSITE for “Creator Comped” (Liaison Technologies) plus an Admin Login that opens a DESKTOP Admin Dashboard (web app) with the same tools/features as the mobile admin UI.

WEBSITE STYLE (premium, modern, clean)
- Sleek black/near-black theme (like Uber)
- SF Pro or modern sans serif
- Liaison gradient (orange → blue) used sparingly for primary CTAs and accents (no heavy glow)
- Lots of whitespace, large typography, rounded cards, subtle shadows
- Desktop-first responsive layout (1440px width), with mobile-friendly responsive behavior

HEADER / NAV (sticky)
- Top-left: Creator Comped logo/wordmark
- Nav links: Features, How it works, Pricing (optional), FAQ, Contact
- Top-right:
  - Primary CTA button: “Get Started” (gradient, zero glow)
  - Text link/button: “Admin Login” (subtle)
- Clicking “Admin Login” opens an Admin Login page (or modal) with email + password.

HERO SECTION
- Headline: “Get your food comped. Create content. Win rewards.”
- Subheadline: short explanation of the app (creators redeem comps, restaurants get content)
- Primary CTA: “Download / Join as Creator” (gradient)
- Secondary CTA: “Join as Restaurant” (neutral)
- Right side: mock device visuals (iPhone mockup showing map discovery + restaurant card + view menu) (placeholder images ok)

FEATURES SECTION (3–6 feature cards)
Include clean icon cards with short copy for each:
1) Discovery map + list (find participating restaurants)
2) Simple ordering with item limits
3) QR code redemption + backup 5-digit code
4) Post submission + verification (links)
5) Monthly leaderboard + prize pool (creators)
6) Business analytics & spend tracking (restaurants)
- Use icons and concise text; keep it professional.

HOW IT WORKS (2 columns)
- “For Creators” steps:
  1) Choose a restaurant
  2) Select menu items (within limits)
  3) Get Comped via QR code
  4) Submit post links
- “For Restaurants” steps:
  1) Set menu + limits
  2) Set deliverables
  3) Employees scan QR
  4) Track spend + analytics

TRUST / SOCIAL PROOF (optional)
- Testimonials placeholders, or “Trusted by local restaurants” logo strip (placeholder)

FOOTER
- Links: Rules, Support, Terms of Service, Privacy Policy
- Liaison Technologies copyright
